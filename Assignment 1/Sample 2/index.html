How to Use the Relational Algebra Visualizer

Getting Started

This tool helps you learn and practice relational algebra operations with visual feedback.

  1. Start by creating tables or loading sample data
  2. Select a table from the dropdown to apply operations
  3. Choose an operation and provide required parameters
  4. View the result and optionally save it as a new table

Creating Tables

  • Enter a unique table name
  • Add column names separated by commas
  • Add data rows with values separated by commas
  • Each row should be on a separate line
  • Use the sample data button for quick testing

Operations Guide

SELECT (σ)

Filters rows based on a condition. Use operators: =, !=, >, >=, <, <=, AND, OR, NOT

Example: GPA > 3.5 AND Major = 'Computer Science'

PROJECT (π)

Selects specific columns from a table. Check the columns you want to keep.

JOIN (⨝)

Combines two tables based on matching values in common columns.

Choose between INNER, LEFT, and RIGHT joins.

UNION (∪)

Combines rows from two tables with compatible schemas.

DIFFERENCE (-)

Finds rows present in the first table but not in the second.

INTERSECT (∩)

Finds common rows between two tables.

Keyboard Shortcuts

  • Ctrl/Cmd + N: Focus on table name input
  • Ctrl/Cmd + D: Load sample data
  • Esc: Clear current result
  • Ctrl/Cmd + H: Clear operation history

Tips & Best Practices

  • Use descriptive table and column names
  • Ensure compatible schemas for set operations
  • Check operation history to track your progress
  • Save important results as tables for future operations
  • Use the toggle view button to focus on specific tables
